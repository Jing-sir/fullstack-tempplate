# 全局状态（GlobalAppLogic）

`GlobalAppLogic` 是应用级单例 Controller，管理登录态、账户信息、网络状态等全局数据。

获取：
```dart
final logic = Get.find<GlobalAppLogic>();
final state = Get.find<GlobalAppLogic>().state;
```

---

## 常用状态字段

### 账户信息

```dart
final state = Get.find<GlobalAppLogic>().state;

// 账户信息（登录后才有值）
final AccountInfo? info = state.accountInfo;
final String email   = info?.email ?? '';
final String userId  = info?.id ?? '';
final String account = info?.userAccount ?? '';

// 安全认证列表
final securityList = state.securityListRep?.list ?? [];
```

### 货币与资产

```dart
// 计价货币列表（Obx 响应式）
Obx(() {
  final currencies = state.fiatCurrencies;
  // ...
})

// 加密货币列表
final coins = state.coins;

// 币种精度（小数位数）
final decimals = state.coinDecimals['BTC'] ?? 8;
```

### 系统开关

```dart
// 资产明文/密文开关（与 CipherText 联动）
final isHidden = state.isCipherText.value;

// 是否暗色主题
final isDark = state.isDark.value;

// 是否联网
final hasNetwork = state.isConnected.value;

// 未读消息数
final unread = state.unreadCount.value;

// 系统功能开关（后端下发）
final sysModel = state.sysModelDisplay.value;
```

### App 生命周期

```dart
Obx(() {
  if (state.appLifecycleState.value == AppLifecycleState.resumed) {
    // App 回到前台
  }
})
```

---

## 常用方法

```dart
final logic = Get.find<GlobalAppLogic>();

// 获取账户信息（自动处理缓存与接口刷新）
final info = await logic.getAccountInfo();

// 获取加解密所需的 link 和 key
final linkAndKey = await logic.linkAndKey();

// 刷新未读消息数
await logic.getUnreadCount();
```

---

## 注意事项

- `state.accountInfo` 是**普通对象**（非 Obs），登录态变化时由接口刷新后直接赋值，需要监听变化用 `GetBuilder<GlobalAppLogic>` 或事件总线的 `LoginEvent` / `LoginOutEvent`
- 不要在页面里直接修改 `state` 的字段，应通过 `logic` 的方法或事件总线触发
- 敏感信息展示用 `CipherText` 组件，它已内置对 `state.isCipherText` 的监听（见空态与敏感信息文档）
