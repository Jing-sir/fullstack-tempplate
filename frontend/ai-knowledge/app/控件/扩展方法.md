# 扩展方法速查

扩展文件位于 `lib/extentions/`，写布局和业务代码时优先使用，避免手动嵌套包裹式写法。

## ex_widget.dart — Widget 链式扩展

导入：`import 'package:flutter_template/extentions/ex_widget.dart';`

### 内边距

```dart
Text('hello').paddingAll(8.w)
Text('hello').paddingHorizontal(16.w)
Text('hello').paddingVertical(12.w)
Text('hello').paddingTop(8.w)
Text('hello').paddingBottom(8.w)
Text('hello').paddingLeft(16.w)
Text('hello').paddingRight(16.w)

// 自定义 EdgeInsets
Text('hello').padding(value: EdgeInsets.fromLTRB(...))
```

### 尺寸约束

```dart
widget.width(100.w)
widget.height(50.w)
widget.tight(width: 100.w, height: 50.w)
widget.tightSize(40.w)          // 正方形
```

### 对齐

```dart
widget.alignCenter()
widget.alignLeft()
widget.alignRight()
widget.alignTop()
widget.alignBottom()
widget.alignTopRight()
widget.alignBottomRight()
widget.center()                 // Center 包裹
```

### 点击手势

```dart
// 带防抖（800ms），推荐用这个
widget.onTap(() => logic.doSomething())

// 长按
widget.onLongPress(() => ...)

// 涟漪效果
widget.inkWell(onTap: () => ..., borderRadius: 8.w)
```

### 裁剪与圆角

```dart
widget.clipRRect(all: 8.w)
widget.clipRRect(topLeft: 8.w, topRight: 8.w)
widget.clipOval()
widget.clipRect()
```

### 布局辅助

```dart
widget.expanded()               // Expanded
widget.expanded(flex: 2)
widget.flexible()
widget.center()                 // Center
widget.opacity(0.5)
widget.backgroundColor(Colors.red)
widget.elevation(4.0)
widget.safeArea()
widget.scrollable()             // SingleChildScrollView 包裹
widget.sliverToBoxAdapter()     // SliverToBoxAdapter 包裹
```

### 装饰

```dart
widget.decorated(
  color: PageStyle.defaultCardColor,
  borderRadius: BorderRadius.circular(12.w),
  boxShadow: [...],
)

widget.border(all: 1.0, color: PageStyle.primaryColor, radius: 8.w)
widget.boxShadow(color: Colors.black26, blurRadius: 8)
```

## ex_widget.dart — List\<Widget\> 扩展

```dart
// 转 Column（支持 separator 自动插入分隔线）
[widget1, widget2].toColumn(
  crossAxisAlignment: CrossAxisAlignment.start,
  separator: Divider(),         // 可选
)

// 转 Row
[widget1, widget2].toRow(
  mainAxisAlignment: MainAxisAlignment.spaceBetween,
)

// 转 Stack
[widget1, widget2].toStack()

// 转 Wrap
[widget1, widget2].toWrap(spacing: 8.w, runSpacing: 8.w)
```

## extension_widget.dart — 补充扩展（PaddingExt）

导入：`import 'package:flutter_template/extentions/extension_widget.dart';`

```dart
widget.addPadding(EdgeInsets.all(8.w))
widget.addGesture(() => ...)
widget.addExpand()
widget.addVisibility(isVisible)
widget.addRadius(8.w)
widget.addContainer(color: ..., radius: 8.0, padding: ...)
```

## ex_decimal.dart — 数字格式化

导入：`import 'package:flutter_template/extentions/ex_decimal.dart';`

```dart
'1234567.89'.permillage()       // → '1,234,567.89'（千分位）
someDecimal.permillage()        // Decimal 类型同上
```

## ex_pwd.dart — 密码加密

导入：`import 'package:flutter_template/extentions/ex_pwd.dart';`

```dart
// 登录态加密（需要 GlobalAppLogic 已初始化）
final encrypted = await password.encrypt();

// 无登录态加密
final encrypted = await password.noLoginEncrypt(key: someKey);

// PCI 三要素解密（CardSimpleInfo / CardCompleteInfo）
final info = await cardSimpleInfo.threeElementsDecrypt(key);
```

## 常见错误

- `.onTap()` 内置了 800ms 防抖，不要再手动包 `GestureDetector` 加防抖逻辑
- `List.toColumn(separator: ...)` 会自动在每两项之间插入分隔符，不需要手动 `expand + removeLast`
- Sliver 场景用 `.sliverToBoxAdapter()` 而不是手动写 `SliverToBoxAdapter(child: widget)`
