# 杂项工具组件

## CustomDivider — 分割线

非 Widget 类，以 getter 形式导出，直接当变量使用，无需构造。

导入：`import 'package:flutter_template/widgets/CustomDivider.dart';`

```dart
// 标准 1dp 水平分割线（列表项之间）
divider

// 浅色变体（比 divider 颜色更浅）
variantDivider

// 第三色水平线（特定区块边界）
dividerF7F8F9

// 垂直分割线（按钮之间竖线）
horizontalDivider

// 8dp 高区块分隔条（页面大区块之间）
divider8H

// 6dp 高区块分隔条（页面中等区块之间）
divider6H
```

典型用法：

```dart
Column(
  children: [
    _itemA(),
    divider,
    _itemB(),
    divider6H,
    _sectionC(),
  ],
)
```

## SliverHeaderDelegate — 吸顶 Header

`SliverPersistentHeaderDelegate` 的封装，用于 `CustomScrollView` / `NestedScrollView` 中的吸顶区域。

导入：`import 'package:flutter_template/widgets/SliverHeaderDelegate.dart';`

### 固定高度吸顶（最常用）

```dart
SliverPersistentHeader(
  pinned: true,
  delegate: SliverHeaderDelegate.fixedHeight(
    height: 44.w,
    child: Container(
      color: Theme.of(context).scaffoldBackgroundColor,
      alignment: Alignment.centerLeft,
      padding: EdgeInsets.symmetric(horizontal: 16.w),
      child: Text('历史记录'.tr, style: PageStyle.headlineTextStyle),
    ),
  ),
)
```

### 可折叠 Header（maxHeight > minHeight）

```dart
SliverPersistentHeader(
  pinned: true,
  delegate: SliverHeaderDelegate(
    maxHeight: 120.w,
    minHeight: 56.w,
    child: _headerWidget(),   // 固定内容，不响应滚动偏移
  ),
)
```

### 动态折叠（响应滚动偏移）

```dart
SliverPersistentHeader(
  pinned: true,
  delegate: SliverHeaderDelegate.builder(
    maxHeight: 120.w,
    minHeight: 56.w,
    builder: (context, shrinkOffset, overlapsContent) {
      final progress = shrinkOffset / (120.w - 56.w);
      return _buildHeader(progress);  // 根据折叠进度调整样式
    },
  ),
)
```

> **调试技巧**：给 child 设置 `key`，debug 模式会自动 print shrinkOffset 和 overlapsContent。

## CountdownView — 倒计时

传入剩余毫秒数，自动每秒递减展示，格式：`DD天 HH : MM : SS`（天=0时隐藏天数）。

导入：`import 'package:flutter_template/widgets/countdown.dart';`

```dart
CountdownView(
  difference: 86400000,   // 单位：毫秒，24小时 = 86400 * 1000
  style: PageStyle.headlineTextStyle!.copyWith(fontSize: 16.sp),
)
```

> **注意**：`difference` 单位是**毫秒**，不是秒。传秒会导致显示时间缩短 1000 倍。
>
> 组件 dispose 时自动取消 Timer，无需手动管理。
