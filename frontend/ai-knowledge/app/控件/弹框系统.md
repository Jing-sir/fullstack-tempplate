# 弹框系统

项目有两个弹框工具类：`BaseDialog`（主要入口）和 `CustomAlert`（补充样式）。

导入：
```dart
import 'package:flutter_template/widgets/base_dialog.dart';
import 'package:flutter_template/widgets/custom_alert.dart';
```

---

## BaseDialog

### showAlertDialog — 通用确认框

单按钮或双按钮，返回 `Future<bool?>`（确认=true，取消=false，点遮罩=null）。

```dart
final result = await BaseDialog.showAlertDialog(
  '确认删除该记录？',
  title: '提示',
  confirm: '确认',         // 默认"确定"
  cancel: '取消',          // 默认"取消"
  canCancel: true,         // false=仅确认按钮（默认），true=左右双按钮
  barrierDismissible: true,
);
if (result == true) { /* 用户点了确认 */ }
```

### showTipDialog — 带关闭图标的提示框

右上角有 × 关闭按钮，单确认按钮，用于说明性文字。

```dart
await BaseDialog.showTipDialog(
  title: '余额不足',
  content: '您的账户余额不足，请充值后重试。',
  confirm: '我知道了',
);
```

### showAlert2Dialog — 样式2确认框

标题居中 + 右上 × 关闭 + 左取消/右确认双按钮布局，更正式的确认场景。

```dart
final result = await BaseDialog.showAlert2Dialog(
  title: '确认提交',
  content: '提交后不可修改，确认继续？',
  cancelStr: '取消',
  sureStr: '确认',
);
```

### showSefDialog — 自定义内容容器

纯壳，传入自定义 child，用于完全自定义弹框内容。

```dart
await BaseDialog.showSefDialog(
  context: Get.context!,
  child: MyCustomWidget(),
  barrierDismissible: false,
);
```

### showTransactionDialog — 3DS 交易确认框

专用于 3DS 安全认证，传入 JSON 字符串（含 last4/transactionAmount/coin/transactionTime/merchant/validateCode）。

```dart
BaseDialog.showTransactionDialog(params: jsonString);
```

### show403ErrorDialog — 403 网络错误提示

单例防重弹，网络层收到 403 响应时调用，不需要 await。

```dart
BaseDialog.show403ErrorDialog();
```

### showSysNetHintDialog — IP 受限地区提示

带"不再提示"勾选，支持持久化到 SharedPreferences。

```dart
BaseDialog.showSysNetHintDialog(
  cardName: '某国家/地区名',
  ditchCardId: cardId,
);
```

### showAdDialog — 广告弹框

由事件总线触发，内部处理频率控制逻辑（每次/每天/自定义间隔），通常不手动调用。

### noticeSwitchDialog — 账户切换确认框

传入 from/to IsarUser 对象，展示切换账户信息。

```dart
final result = await BaseDialog.noticeSwitchDialog(
  formUser: currentUser,
  toUser: targetUser,
);
```

## CustomAlert

补充样式弹框，圆角更大（20/8）。

### info — 带顶部图片横幅 + 单按钮

```dart
CustomAlert.info(
  context,
  title: '标题',
  url: 'assets/images/banner.png',   // 顶部图片，可不传
  contentWeight: Text('内容'),        // 必填
  confirmButtonText: '确定',
  onConfirm: () => Navigator.pop(context),
);
```

### infoTwo / infoTwoNew / infoNewTwo — 双按钮变体

三者逻辑相同，仅按钮宽度和图片资源略有差异。推荐用 `infoNewTwo`（最新）：

```dart
CustomAlert.infoNewTwo(
  context,
  title: '提示',
  contentWeight: Text('内容'),
  confirmButtonText: '确定',
  cancelButtonText: '取消',
  canCancel: true,              // false=仅确认按钮
  onConfirm: () { ... },
  onCancel: () => Navigator.pop(context),
);
```

### infoNew — 带关闭图标 + 单按钮（新样式）

```dart
CustomAlert.infoNew(
  context,
  title: '提示',
  contentWeight: Text('内容'),
  confirmButtonText: '确定',
  onConfirm: () { ... },
);
```

## 选型指南

| 场景 | 推荐 |
| --- | --- |
| 简单文字确认（单/双按钮） | `BaseDialog.showAlertDialog` |
| 说明性文字，用户需阅读 | `BaseDialog.showTipDialog` |
| 需要标题+内容+双按钮（正式） | `BaseDialog.showAlert2Dialog` |
| 完全自定义内容 | `BaseDialog.showSefDialog` |
| 带顶部图片的营销/引导弹框 | `CustomAlert.info` / `infoNew` |
| 双按钮确认（自定义内容区） | `CustomAlert.infoNewTwo` |
| 全局 loading 遮罩 | `EasyLoading.show()` / `.dismiss()` |
