# 本地存储（SpUtils）

本项目统一使用 `SpUtils` 封装 `SharedPreferences`，**不要直接调用** `SharedPreferences.getInstance()`。

导入：
```dart
import 'package:flutter_template/utils/sp_utils.dart';
import 'package:flutter_template/config/sp_config.dart';
```

---

## 基本用法

```dart
// 写入
SpUtils.getInstance().setData(SpConfig.locale, 'zh_CN');
SpUtils.getInstance().setData(SpConfig.themeType, 1);      // int
SpUtils.getInstance().setData(SpConfig.user_token, token); // String
SpUtils.getInstance().setData('someFlag', true);           // bool

// 读取（需要指定泛型）
final token = SpUtils.getInstance().get<String>(SpConfig.user_token);
final theme  = SpUtils.getInstance().get<int>(SpConfig.themeType);
final flag   = SpUtils.getInstance().get<bool>('someFlag') ?? false;

// 删除单个
SpUtils.getInstance().remove(SpConfig.user_token);
```

---

## Key 规范（SpConfig）

所有 key 定义在 `lib/config/sp_config.dart`，**禁止硬编码字符串**，必须在 `SpConfig` 里声明后使用。

| Key 常量 | 类型 | 说明 |
| --- | --- | --- |
| `user_token` | String | 登录 token |
| `user_id` | String | 用户 ID |
| `user_account` | String | 用户账号 |
| `fcm_token` | String | FCM 推送 token |
| `locale` | String | 当前语言，如 `zh_CN` |
| `themeType` | int | 主题类型（见主题文档） |
| `currency` | String | 计价货币 (`invoiceCurrency`) |
| `firstLogin` | bool | 是否首次登录 |
| `isShowGuide` | bool | 是否已显示引导页 |
| `patternLock` | String | 手势密码轨迹 |
| `biometricLock` | bool | 生物识别开关 |
| `lockTime` | int | 锁屏时间设置 |
| `adDialogId` | String | 最近展示的广告 ID |
| `adDialogTime` | int | 广告展示时间戳 |
| `reapHint` | String | 不再提示的 IP 受限卡 ID 列表 |

> API 缓存 key 直接用 `APIs.xxx`（如 `APIs.accountInfo`），不在 SpConfig 定义。

---

## 登出时清理

登出时调用 `SpUtils.getInstance().logout()` 即可，内部已清理 token、账号、API 缓存等所有登录相关数据，**不需要手动逐个 remove**。

```dart
await SpUtils.getInstance().logout();
```

---

## 注意事项

- `setData` 是**同步写入**（SharedPreferences 内部异步，但无需 await）
- `get<T>` 返回 `T?`，读取时注意判空或提供默认值
- 新增 key 前先搜索 `SpConfig` 确认没有同语义的已有 key
