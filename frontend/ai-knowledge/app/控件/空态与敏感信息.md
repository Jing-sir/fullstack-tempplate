# 空态与敏感信息

## NoData — 空态占位图

纯展示组件，无状态管理，直接在 `children` 里使用。

导入：`import 'package:flutter_template/widgets/no_data.dart';`

```dart
// 大图 + 文字（默认，列表为空时）
NoData()

// 自定义文字
NoData(noTitle: '暂无记录'.tr)

// 仅小图标（嵌入行内使用）
NoData(isImage: true)

// 自定义图片
NoData(imageUrl: 'assets/images/custom_empty.png')
```

## NoDataView — 异步加载 + 空态容器

封装"加载中 / 有数据 / 空态"三种状态，适合**不使用 ShimmerLoadingView 的简单列表页**。

导入：`import 'package:flutter_template/widgets/no_data_view.dart';`

```dart
NoDataView(
  loading: logic.fetchData,    // Future Function()，initState 自动调用
  hasData: state.list.isNotEmpty,
  child: Obx(() => ListView(
    children: state.list.map((e) => _item(e)).toList(),
  )),
  noTitle: '暂无数据'.tr,       // 可选
)
```

**加载中**显示 `LoadingPage`（旋转指示器），**有数据**显示 `child`，**无数据**显示 `NoData`。

> **注意**：`hasData` 在 `loading` 完成后才被读取。如果用 GetX 响应式变量，确保 `hasData` 表达式在重建时能正确取到最新值。

### 与 ShimmerLoadingView 的区别

| | `NoDataView` | `ShimmerLoadingView` |
| --- | --- | --- |
| 加载态样式 | 旋转圈 | 骨架屏 Shimmer |
| 适用场景 | 简单列表、弹框内列表 | 全页首屏加载 |
| 骨架屏 | 不需要 | 需要配套 `skeleton_xxx.dart` |

## CipherText — 敏感信息脱敏展示

余额、卡号等敏感字段统一用此组件，与全局隐私开关联动。

导入：`import 'package:flutter_template/widgets/cipher_text.dart';`

```dart
// 跟随全局隐私开关（推荐）
CipherText(
  text: state.balance.value,
  style: PageStyle.headlineTextStyle?.copyWith(fontSize: 24.sp),
)

// 页面单独控制
CipherText(
  text: state.cardNumber.value,
  isCipherText: state.isHidden.value,  // true=显示****，false=显示原文
  style: PageStyle.ts_111A34_14sp,
)
```

**触发 `****` 的条件：**
1. 传了 `isCipherText: true`
2. 未传 `isCipherText` 且 `GlobalAppLogic.state.isCipherText.value == true`（全局隐私开关）
3. 用户未登录（`LiteRouteConfig.checkLogin()` 返回 true）

> **不要**用 `Text(state.isCipherText ? '****' : balance)` 自己实现脱敏，用 `CipherText` 保证与全局开关同步。
