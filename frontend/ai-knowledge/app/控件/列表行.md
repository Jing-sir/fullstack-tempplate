# CustomListTile — 列表行

设置页、表单详情页中"标题 + 右侧值/箭头"的标准行组件。

导入：`import 'package:flutter_template/widgets/custom_list_tile.dart';`

---

## 基本用法

```dart
// 纯展示行（无箭头、无点击）
CustomListTile(
  title: '姓名'.tr,
  trailing: state.name.value,
)

// 可点击行（自动显示右箭头）
CustomListTile(
  title: '收货地址'.tr,
  trailing: state.address.value,
  onTap: () => logic.goAddress(),
)

// 隐藏箭头但保留点击
CustomListTile(
  title: '复制'.tr,
  hideArrow: false,   // false = 显示箭头；true(默认) = 隐藏箭头
  onTap: () => logic.copy(),
)
```

---

## 常用参数

| 参数 | 类型 | 说明 |
| --- | --- | --- |
| `title` | String | 左侧标签文字（必填） |
| `trailing` | String? | 右侧纯文字值 |
| `trailingWidget` | Widget? | 右侧自定义 Widget（优先于 trailing） |
| `onTap` | VoidCallback? | 点击回调，有值时自动显示箭头 |
| `hideArrow` | bool | 默认 `true`（隐藏箭头）；`false` 强制显示 |
| `iconUrl` | String? | 标题左侧图标（asset 路径） |
| `mustStyle` | bool | `true` = 标题前显示红色 `*` |
| `disabled` | bool | `true` = 禁用点击（灰色） |
| `padding` | EdgeInsets? | 默认 `vertical: 12.w` |
| `titleTextStyle` | TextStyle? | 覆盖标题样式 |
| `trailingTextStyle` | TextStyle? | 覆盖右侧文字样式 |
| `bottomTip` | String? | 行底部小提示文字 |

## 自定义右侧内容

```dart
// 右侧放 Switch
CustomListTile(
  title: '消息通知'.tr,
  trailingWidget: Obx(() => CupertinoSwitch(
    value: state.notifyEnabled.value,
    onChanged: (v) => logic.toggleNotify(v),
    activeColor: PageStyle.primaryColor,
  )),
)

// 右侧放状态标签
CustomListTile(
  title: '认证状态'.tr,
  trailingWidget: Container(
    padding: EdgeInsets.symmetric(horizontal: 8.w, vertical: 2.w),
    decoration: BoxDecoration(
      color: PageStyle.primaryColor.withOpacity(0.1),
      borderRadius: BorderRadius.circular(4.w),
    ),
    child: Text('已认证'.tr, style: PageStyle.ts_primary_12sp),
  ),
)
```

## 从 TiLeItem 构建（批量列表场景）

`TiLeItem` 是数据模型，适合把一组设置项定义为列表再统一渲染：

```dart
// 定义数据
final items = [
  TiLeItem(title: '修改密码'.tr, onTap: logic.goChangePassword),
  TiLeItem(title: '绑定邮箱'.tr, label: state.email.value, onTap: logic.goBindEmail),
  TiLeItem(title: '注销账户'.tr, onTap: logic.goDeleteAccount),
];

// 渲染
Column(
  children: items.map((item) {
    return Column(children: [
      CustomListTile.tiLeItem(item),
      divider,
    ]);
  }).toList(),
)
```
