# 按钮组件规范

## 结论：统一使用 CommonButton

页面内所有按钮（主按钮、次按钮、描边按钮）必须使用 `CommonButton`（`lib/widgets/common_button.dart`），**不得**直接使用 Flutter 原生的 `ElevatedButton`、`OutlinedButton`、`TextButton`。

---

## CommonButton 参数速查

```dart
CommonButton({
  String? buttonContent,          // 按钮文字
  Widget? buttonContentWidget,    // 自定义内容（优先于 buttonContent，用于 loading 态）
  double? buttonHeight,           // 默认 48.w
  double? buttonWidth,            // 默认 double.infinity（撑满父容器）
  bool autoWidth = false,         // true = 宽度自适应文字
  bool autoHeight = false,        // true = 高度自适应内容
  bool disabled = true,           // ⚠️ true=可点击，false=禁用（语义与字面相反）
  VoidCallback? onPressed,        // 点击回调
  Color? normalColor,             // 背景色，默认 PageStyle.primaryColor
  Color? borderColor,             // 描边颜色（配合透明背景实现描边按钮）
  TextStyle? textStyle,           // 文字样式，默认 PageStyle.ts_ffffff_14sp
  EdgeInsetsGeometry? margin,     // 外边距
  int milliseconds = 800,         // 防重复点击间隔（ms）
  double? borderRadius,           // 圆角，默认 8.w
})
```

## 常见用法

### 主按钮（默认主题色）

```dart
CommonButton(
  buttonContent: '保存设置',
  onPressed: logic.save,
)
```

### 禁用态（不可点击，自动变灰）

```dart
// disabled: false → 禁用，自动显示灰色（浅色模式 0xFFC5CAD5，深色模式 0xFF555555）
CommonButton(
  buttonContent: '保存设置',
  disabled: false,
  onPressed: logic.save,
)
```

### 响应式禁用（配合 GetX Obx）

```dart
// 通过 Obx 包裹，让 disabled 跟随 state 变化
Obx(() => CommonButton(
  buttonContent: '保存设置',
  disabled: state.canSave,   // canSave 是 state 里的计算属性
  onPressed: logic.save,
))
```

### 按钮内嵌 loading（保存中）

```dart
Obx(() => CommonButton(
  disabled: state.canSave,   // 保存中时 canSave 返回 false，按钮自动禁用
  buttonContentWidget: state.isSaving.value
      ? const SizedBox(
          width: 18,
          height: 18,
          child: CircularProgressIndicator(strokeWidth: 2, color: Colors.white))
      : null,
  buttonContent: '保存设置',  // buttonContentWidget 为 null 时回退到文字
  onPressed: logic.save,
))
```

### 描边次按钮（无背景色）

```dart
CommonButton(
  buttonContent: '查看通知记录',
  normalColor: Colors.transparent,
  borderColor: Colors.grey,
  textStyle: const TextStyle(color: Colors.black87),
  onPressed: logic.goRecords,
)
```

## ⚠️ disabled 语义说明

`CommonButton` 的 `disabled` 参数**语义与字面相反**：

| `disabled` 值 | 实际效果 |
| --- | --- |
| `true`（默认值） | **可点击** |
| `false` | **禁用**，自动变灰，onPressed 不触发 |

这是项目历史遗留设计，开发时需特别注意，不要按字面意思理解。

## 错误示范（不要这样写）

```dart
// ❌ 不要直接用原生按钮
ElevatedButton(
  onPressed: logic.save,
  child: Text('保存'),
)

// ❌ 不要用 OutlinedButton
OutlinedButton(
  onPressed: logic.goBack,
  child: Text('返回'),
)
```

## 底部按钮的安全距离与高度

### 标准做法

底部按钮**不能**放在 `body` 的 Column 末尾，必须放在 `Scaffold.bottomNavigationBar`，并用 `SafeArea(bottom: true)` 包裹，确保 iPhone 底部 Home Indicator 区域不被遮挡。

```dart
Scaffold(
  appBar: ...,
  bottomNavigationBar: SafeArea(
    bottom: true,
    child: _buttons().paddingAll(16.w), // 统一 16.w 内边距
  ),
  body: ...,
)
```

### 按钮高度

| 场景 | 高度 |
| --- | --- |
| 单按钮 / 独占一行 | `buttonHeight: 48.w` |
| 多按钮横向并排 | `buttonHeight: 40.w` |

高度必须用 `flutter_screenutil` 的 `.w` 单位，**不要用固定 px 或 `EdgeInsets` 的 vertical padding 代替**。

### 完整示例

#### 单按钮

```dart
bottomNavigationBar: SafeArea(
  bottom: true,
  child: CommonButton(
    buttonContent: '确认',
    buttonHeight: 48.w,
    onPressed: logic.confirm,
  ).paddingAll(16.w),
),
```

#### 多按钮（主 + 次）

多按钮必须用 `Column(mainAxisSize: MainAxisSize.min)` 包裹，防止 Column 撑满整个 `bottomNavigationBar` 高度。

```dart
bottomNavigationBar: SafeArea(
  bottom: true,
  child: Column(
    mainAxisSize: MainAxisSize.min, // ⚠️ 多按钮必须加，否则 Column 撑满底栏高度
    children: [
      CommonButton(
        buttonContent: '保存设置',
        buttonHeight: 48.w,
        onPressed: logic.save,
      ),
      SizedBox(height: 12.w),
      CommonButton(
        buttonContent: '取消',
        buttonHeight: 48.w,
        normalColor: Colors.transparent,
        borderColor: Colors.grey,
        textStyle: TextStyle(color: Colors.black87),
        onPressed: Get.back,
      ),
    ],
  ).paddingAll(16.w),
),
```

### 错误示范

```dart
// ❌ 不要放在 body 的 Column 末尾
body: Column(
  children: [
    Expanded(child: content),
    Padding(
      padding: const EdgeInsets.all(16),
      child: CommonButton(...),  // 缺少安全距离
    ),
  ],
),

// ❌ 不要用 padding vertical 代替 buttonHeight
CommonButton(
  padding: EdgeInsets.symmetric(vertical: 14), // 错误
  ...
),
```
