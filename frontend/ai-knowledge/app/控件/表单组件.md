# 表单组件

## CommonEditBox — 通用输入框

带标题行的表单输入框，下划线样式，自动验证。

导入：`import 'package:flutter_template/widgets/CommonEditBox.dart';`

```dart
CommonEditBox(
  controller: state.amountController,
  title: '金额'.tr,            // 可不传，不传则无标题行
  hint: '请输入'.tr,
  mustStyle: true,            // true=标题前显示红色 *
  enable: true,               // false=禁用（只读）
  obscureText: false,         // true=密文（密码场景）
  keyboardType: TextInputType.numberWithOptions(decimal: true),
  inputFormatters: [
    FilteringTextInputFormatter.allow(RegExp(r'^\d+\.?\d{0,2}')),
  ],
  prefix: Text('USD', style: PageStyle.headlineTextStyle),  // 前缀 Widget
  suffix: Icon(Icons.clear),                                 // 后缀 Widget
  titleEndWidget: TextButton(child: Text('全部'.tr), onPressed: () {}), // 标题行右侧
  validator: (v) => v!.isEmpty ? '不能为空'.tr : null,
  onChanged: (v) => state.amount.value = v,
  maxLength: 20,
)
```

**注意：**
- 内部使用 `autovalidateMode: AutovalidateMode.onUserInteraction`，用户输入后自动触发校验
- 如需要 Form 级别整体校验，外层包 `Form` + `GlobalKey<FormState>`，调用 `formKey.currentState!.validate()`

## CustomDropdownButton — 下拉选择框

泛型组件，数据项必须继承 `GenderItem`。

导入：`import 'package:flutter_template/widgets/custom_dropdown_button.dart';`

### 1. 定义数据模型

```dart
class CountryItem extends GenderItem {
  final String name;
  final String code;

  CountryItem({required this.name, required this.code});

  @override
  String? getValueStr() => name;   // 列表中显示的文字

  @override
  String? getValueId() => code;    // 实际值（传给接口）
}
```

### 2. 默认样式（标题+选中值横排）

```dart
CustomDropdownButton<CountryItem>(
  title: '国家/地区'.tr,
  hint: '请选择'.tr,
  mustStyle: true,
  genderItems: state.countryList,
  value: state.selectedCountry,
  valueChanged: (item) {
    state.selectedCountry = item as CountryItem;
  },
)
```

### 3. 值右对齐变体

```dart
CustomDropdownButton<CountryItem>.valueRight(
  title: '国家/地区'.tr,
  hint: '请选择'.tr,
  genderItems: state.countryList,
  value: state.selectedCountry,
  valueChanged: (item) { ... },
)
```

### 4. 无标题变体

```dart
CustomDropdownButton<CountryItem>.noTitle(
  hint: '请选择'.tr,
  genderItems: state.countryList,
  value: state.selectedCountry,
  valueChanged: (item) { ... },
)
```

## CustomDropdownButton2 — 下拉框（下划线样式）

与 `CustomDropdownButton` 逻辑相同，样式改为下划线（与 `CommonEditBox` 一致），适合表单页。

导入：`import 'package:flutter_template/widgets/custom_dropdown_button2.dart';`

```dart
CustomDropdownButton2<CountryItem>(
  title: '国家/地区'.tr,
  hint: '请选择'.tr,
  genderItems: state.countryList,
  value: state.selectedCountry,
  valueChanged: (item) { ... },
)
```

## 选型指南

| 场景 | 使用 |
| --- | --- |
| 卡片/独立选择区 | `CustomDropdownButton`（圆角边框） |
| 表单页，与 CommonEditBox 同列 | `CustomDropdownButton2`（下划线） |
