# 项目概览

## 基础信息

- 项目目录：`/Users/kong/StudioProjects/upay_app`
- 应用品牌：UP Business
- Flutter 包名（`pubspec.yaml`）：`flutter_template`
- 当前版本（`pubspec.yaml`）：`1.7.8+785`
- Flutter 约束：`^3.32.8`
- Dart 约束：`>=3.0.0 <4.0.0`

## 技术栈

- UI 与状态管理：Flutter + GetX（`get`）
- 网络层：Dio（含自定义拦截器）
- 本地存储：`shared_preferences`、Isar
- 多语言：`flutter_localizations` + `lib/language/*`
- 推送与统计：Firebase Messaging / Analytics / Crashlytics
- 其它关键能力：`app_links`（深链）、`local_auth`（生物识别）、`flutter_smart_dialog`、`flutter_easyloading`

## Flavor 与运行入口

入口文件位于 `lib/main_*.dart`：

- `lib/main_qa.dart`
- `lib/main_prod.dart`
- `lib/main_google.dart`
- `lib/main_huawei.dart`

运行时主要差异：

1. `F.appFlavor` 设置（`lib/flavors.dart`）。
2. `AppConfig.API_HOST` 指向测试或生产。
3. Firebase options 切换（QA / Prod）。

## 业务域（按目录）

核心业务页面集中在 `lib/lite_page/`，主要模块包括：

- 资产（asset）
- 卡（card）
- 市场（market）
- 汇款（remit / no_cause_remit）
- 质押借贷（pledge）
- 理财（financial_management）
- 安全（security）
- 个人中心（mine）

历史或补充页面位于 `lib/pages/` 与 `lib/new_pages/`。
