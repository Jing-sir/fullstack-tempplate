# 本地开发指南

## 环境要求

- Flutter：`3.32.8`（见 `pubspec.yaml`）
- Dart：`>=3.0.0 <4.0.0`
- Android：`compileSdk 36`、`minSdk 28`（见 `android/app/build.gradle`）
- iOS：需本机安装 Xcode 与 CocoaPods

## 首次初始化

1. 拉取仓库到本地。
2. 配置 `opc` 依赖目录（来自根 `README.md` 说明）：
3. 在 `android/local.properties` 加入：`opc.repo=本地路径/repo`
4. 执行依赖安装：`flutter pub get`

## 常用运行命令

QA 环境：

```bash
flutter run --flavor qa -t lib/main_qa.dart
```

生产环境（prod）：

```bash
flutter run --flavor prod -t lib/main_prod.dart
```

Google 渠道：

```bash
flutter run --flavor google -t lib/main_google.dart
```

Huawei 渠道：

```bash
flutter run --flavor huawei -t lib/main_huawei.dart
```

## 常见本地问题

1. `upay_opc` 拉取失败：检查 SSH Key 与对 `git@github.com:upayProject/WalletCardAdd.git` 的访问权限。
2. iOS 编译失败：执行 `cd ios && pod install` 后重试。
3. 依赖缓存异常：可尝试 `flutter clean && flutter pub get`。
4. 登录态异常：检查 `SharedPreferences` 中 `user_token` 是否残留。
