# 项目技术栈

## 1. 项目形态

- Maven 多模块微服务项目（根 `pom.xml` 聚合）。
- 根模块下共 `26` 个一级模块。
- 代码组织以 `*-service`（服务实现）+ `*-client`（服务接口/调用）为主，扫描到 `14` 个 `service` 子模块与 `14` 个 `client` 子模块。
- 主开发语言为 Java（扫描到 `8416` 个 `.java` 文件，未发现前端 Node/Vue/TS 工程特征）。

## 2. 核心技术栈

### 2.1 语言与构建

- Java `1.8`（多处 `maven-compiler-plugin` `source/target=1.8`）。
- 构建工具：Maven（根与子模块均为 `pom.xml` 体系）。
- Spring Boot 版本混用：
    - `2.3.1.RELEASE`
    - `2.3.3.RELEASE`

### 2.2 基础框架

- Spring Boot（`spring-boot-starter`、`spring-boot-starter-web`）。
- Spring Cloud：`Hoxton.SR3`（`spring-cloud-dependencies`）。
- Spring Cloud Alibaba Nacos：
    - `spring-cloud-starter-alibaba-nacos-discovery:2.0.0.RELEASE`
    - `spring-cloud-starter-alibaba-nacos-config:0.9.0.RELEASE`
- RPC：Apache Dubbo `2.7.3`（`dubbo-spring-boot-starter`、`dubbo-registry-nacos`）。

### 2.3 数据与存储

- MySQL：`mysql-connector-java`。
- ORM/持久层：
    - `mybatis-plus-boot-starter`
    - `mybatis-spring-boot-starter`
    - `pagehelper`
- 连接池：`druid-spring-boot-starter`。
- Redis：
    - `spring-boot-starter-data-redis`
    - `jedis`
    - `commons-pool2`
- Elasticsearch（部分模块）：
    - `elasticsearch`
    - `elasticsearch-rest-high-level-client`
    - `spring-data-elasticsearch`

### 2.4 消息与异步

- RabbitMQ（AMQP）：`spring-boot-starter-amqp`。
- 配置侧也多处引用 `rabbit.yml`（通过 Nacos 外部配置共享）。

### 2.5 文档与接口治理

- Swagger 体系：
    - `springfox-swagger2`
    - `springfox-swagger-ui`
    - `swagger-bootstrap-ui`

### 2.6 常见基础库与业务能力

- `lombok`
- `fastjson`
- `spring-boot-starter-mail`（邮件能力）
- `lock4j-redis-template-spring-boot-starter`（分布式锁）
- `webauthn4j-core`（WebAuthn）
- `geoip2`（IP 地理解析）
- `jxls` / `poi-ooxml`（Excel 报表导出）
- `retrofit` / `okhttp`（HTTP 客户端）

### 2.7 对象存储与云生态集成

- AWS：
    - `aws-java-sdk-s3`
    - `aws-java-sdk-sns`
    - `software.amazon.awssdk:sns`
- 阿里云：
    - `aliyun-sdk-oss`
    - `aliyun-java-sdk-core`
    - `aliyun-java-sdk-dm`
- MinIO：`io.minio:minio`

## 3. 配置与环境管理

- 多环境配置命名规范统一：`bootstrap-dev/test/pre/prod.yml`。
- 多模块通过 Nacos 动态装配共享配置，常见 `shared-dataids` 包括：
    - `mysql.yml`
    - `redis.yml`
    - `rabbit.yml`
    - `aws-s3.yml`
    - 以及各业务域配置（如 `user.yml`、`monitor.yml` 等）。
- i18n 资源存在，支持多语言属性文件（如 `zh-CN`、`en-US` 等）。

## 4. 部署与运行

- 多服务含 `docker/Dockerfile`。
- Docker 基础镜像统一为：`primetoninc/jdk:1.8`。

## 5. 模块分层（按命名归纳）

- 基础共享：`upay-model`、`upay-common`、`upay-asset`、`upay-open-sdk`
- API/接入层：`upay-app-api`、`upay-open-api`、`upay-broker-api`、`upay-caas-api`、`upay-external-api`、`upay-sys-api`
- 领域服务层：`upay-user-server`、`upay-wallet-server`、`upay-card-server`、`upay-borrow-server`、`upay-swap-server`、`upay-broker-server`、`upay-earn-server`、`upay-ditch-server`、`upay-acquirer-server`、`upay-notice-server` 等
- 任务与聚合：`upay-task`、`upay-sys-aggregation-server`、`upay-migrate-server`

## 6. 结论

该项目是典型的 Java 8 + Spring Boot + Spring Cloud Alibaba + Dubbo 的金融业务微服务体系，核心中间件为 Nacos、MySQL、Redis、RabbitMQ，并在部分业务中引入 Elasticsearch 与多云对象存储能力。
